About

I'm a theoretical physicist (Master at UCLouvain, graduated in 2007), eager to learn and passionate about teaching. I love learning about Web technologies and Data Science.

I've been doing web development since 1999, out of passion. Back in 1999, I started with plain HTML/CSS and the Notepad editor, then moved to PHP/MySQL. I then learned Wordpress, how to create plugins and customize Wordpress themes, how to integrate WP with other systems using webservices, etc. Today, I'm getting up to date with the latest technologies in Front-/Back-end web developments: Bootstrap, CSS grid, Angular, jQuery, NodeJS, ReactJS, etc. On the other hand, I've been learning Data Science techniques (see here).

I'm currently working as a technical/functional trainer at Sopra Banking Software.

I also play the violin by the way, not that it has to do with anything :-)
